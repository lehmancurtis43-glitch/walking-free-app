Data Sharing

Walking Free does not share any user data with third parties.

The only third-party services we use are:

  • Clerk — Authentication service. Your email is used only for login. Clerk does not use your data for advertising. Privacy Policy
  • Google Play — App distribution platform. Privacy Policy

Data Deletion

Users can delete their account and all associated data through the Settings page in the app. Data is permanently removed within 30 days.

Security Practices

  • All data transmitted between the app and our servers uses TLS encryption (HTTPS)
  • User passwords are never stored in plain text (handled by Clerk)
  • Local storage data stays on the device and is never transmitted when the user is not signed in
  • Database access is restricted to authenticated users only

Google Play Console — Step-by-Step

1. Does your app collect or share any of the required user data types?

Select: Yes

2. Is all user data encrypted in transit?

Select: Yes

3. Do you provide a way for users to request that their data is deleted?

Select: Yes

4. Data Types

Check these boxes:

  • Email address — Collected, Not shared, App functionality
  • Personal info (sobriety date, name) — Collected, Not shared, App functionality
  • User-generated content (journal entries) — Collected, Not shared, App functionality
  • App interactions (crash data) — Collected, Not shared, Analytics
  • Device or other IDs — Collected, Not shared, App functionality

5. Data Sharing

Does your app share data with any third parties? Select No

6. Security Practices

Check these:

  • Data is encrypted in transit
  • You can request data deletion

7. Data Handling for Account Required Apps

Account creation is optional. Users can use the app without signing in. When signed in, data is synced across devices.
